# Padrões do Projeto

Esta seção reúne os padrões definidos para organização do desenvolvimento do sistema.

Aqui são documentados padrões de commits, pull requests, Kanban e organização geral do projeto.

## Objetivos

- Padronizar práticas de desenvolvimento
- Organizar o fluxo de contribuição do time
- Garantir consistência no repositório
- Melhorar legibilidade e manutenção do código
