# Dashboard de Produtividade

Esta seção apresenta o dashboard de produtividade da equipe integrado à documentação do projeto.

O dashboard exibe métricas relacionadas ao desenvolvimento, contribuições e evolução do projeto.

## Acesso ao Dashboard

- [Abrir Dashboard](dashboard.html)

## Estrutura

- `dashboard.html`: interface visual do dashboard
- `metrics.json`: dados utilizados para geração das métricas

## Objetivo

Centralizar visualizações de produtividade e permitir futuras expansões da plataforma de monitoramento.
