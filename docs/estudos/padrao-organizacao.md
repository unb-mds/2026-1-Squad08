## padrão de organização

### pastas

- usar sempre letras minúsculas
- evitar acentos
- usar hífen para separar palavras

ex:

git-github

metodos-ageis

requisitos

### arquivos

- seguir o mesmo padrão das pastas
- nome simples e direto, de acordo com o conteúdo

ex:

requisitos-software.md

comandos-git.md

### organização geral

* cada assunto na sua pasta
* evitar deixar arquivo solto
* manter o mesmo padrão sempre que criar algo novo
